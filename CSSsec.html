What is CSS

Cascading Style Sheets (CSS) is a stylesheet language used for describing the presentation of a document written in HTML. It describes how elements are to be displayed on the screen, in print, or in other media.

It allows for the separation of content and presentation, making it easier to control the layout and design of a web page.

  1. Styling text and fonts
  2. Setting background colors and images
  3. Defining layout and positioning
  4. Creating responsive designs
